# Mental Health Analysis Among Students

## 📊 Project Overview
This project analyzes mental health patterns among university students using Excel. It explores depression, anxiety, panic attacks and help-seeking behavior.

## 🎯 Objectives
- Identify which gender is most affected by depression
- Analyze which age group has the highest anxiety levels
- Determine percentage of students seeking mental health support
- Compare CGPA with panic attack occurrences

## 🛠 Tools Used
- Microsoft Excel
- Pivot Tables
- Charts (Bar, Pie, Column)

## 📁 Files in this Repository
- Excel dataset and analysis file
- Final project report
- Charts and visualizations

## 📈 Key Insights
- Depression levels varied by gender
- Students aged 18 showed higher anxiety levels
- Only a small percentage of students sought help
- Panic attacks were more common among average-performing students
